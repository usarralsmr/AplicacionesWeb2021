Definiciones importantes

HTML

Son las siglas de HiperText Markup Languaje (Lenguaje de Marcación de Hipertexto) y es un lenguaje que se utiliza comúnmente para establecer la estructura y contenidos de un sitio web, tanto de texto, objetos e imagenes

CSS

Son las siglas de Cascading Style Sheets - Hojas de Estilo en Cascada - que es un lenguaje que describe la presentación de los documentos estructurados en hojas de estilo para diferentes métodos de interpretación

Javascript

Es un lenguaje de programación interpretado. Se define como orientado a objetos, basado en prototipos, imperativo, debilmente tipado y dinámico

"La etiqueta %lt; b %gt; pone el texto en negrita"
